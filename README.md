# startup-funding-analysis  
Analyzing global tech startup trends from 2000–2022 using Google Sheets, BigQuery (SQL), and Tableau.

This project follows the 6-phase Google Data Analytics framework to explore funding patterns, top industries, and emerging startup hubs worldwide.

📁 Includes:  
- Cleaned & analyzed SQL queries  
- Final report PDF & documentation  
- Visuals + story-driven charts
- 
---

## 📊 Tools Used

- **BigQuery (SQL)** for querying and cleaning data  
- **Tableau** for building dashboards and visual storytelling  
- **Google Sheets / Excel** for pre-cleaning  
- **Markdown** and documentation tools for reporting

---

## 📄 Project Highlights

- 🔍 Cleaned and analyzed a real-world dataset of 5,000 startups from Kaggle  
- 🌍 Found surprising leaders like **Germany and Brazil** topping startup density  
- 📦 Uncovered that **Series C** stage had the highest average funding  even more than IPOs  
- 🧠 Built visualizations to showcase trends in funding, geography, and industry


## 📎 Bonus

This project applies the **Google Data Analytics Certificate**'s 6-step methodology:
1. **Ask** – Defined the business problem and goals  
2. **Prepare** – Explored the dataset, handled missing values, and assessed bias  
3. **Process** – Cleaned and standardized all key columns  
4. **Analyze** – Ran SQL queries to generate insights  
5. **Share** – Built clean, interactive charts in Tableau  
6. **Act** – Delivered actionable recommendations for VCs and ecosystem players
